# 資料庫監控工具 DB Monitor

此專案主要是為了在圖表化空氣感測器數據，
可以產生。

## 功能

- 依時間區段選擇指定時段監測數據
- 針對感測器篩選所需資料
- 產生數據分析結果檔案並可供下載

## 專案架構

- 此專案實現部份為 Web Application -> Backend Server -> Database 部份

![](https://github.com/s951080603/DB-Monitor/blob/main/files/architecture.png?raw=true)

## 使用技術

- 前端： JavaScript + React
- 後端： Node.js + Express
- 資料庫： PostgreSQL
